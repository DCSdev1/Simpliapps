09 · Release 1.2.1

Lo nuevo en mySupli 1.2.1

Novedades visibles

New

Los suplementos multiples ya tienen soporte visible en la app, con creacion y edicion mas claras para rutinas y complejos.

New

La importacion CSV mejora para distinguir rutinas y complejos, con una experiencia mas fiable al traer composiciones estructuradas.

New

El historial agrupa mejor las tomas y refleja con mas precision lo que el usuario consume a lo largo del dia.

New

Progreso, racha y adherencia muestran una lectura mas realista del seguimiento diario, especialmente con suplementos programados.

Mejoras de experiencia

Fix

La logica de frecuencias y ciclos ahora se comporta de forma mas coherente entre Home, Historial y Progreso.

Fix

Catalogo, busqueda y localizacion en espanol e ingles reciben ajustes para que encontrar suplementos sea mas rapido y natural.

Fix

La Home gana jerarquia visual y una experiencia mas pulida para acciones rapidas como la toma puntual.

Ready

Esta release deja una base mas solida para seguir ampliando mySupli sin perder simplicidad en el uso diario.
