Privacidad de la web

Esta pagina explica como se tratan los datos personales cuando navegas por el sitio web de Simpli o utilizas el formulario de contacto.

Ultima actualizacion: 23 de marzo de 2026 Responsable: Daniel Carrion Santiago Contacto: dcarrion.desarrollo@gmail.com

1. Responsable del tratamiento

El responsable del tratamiento de los datos personales recabados a traves de este sitio web es Daniel Carrion Santiago. Si tienes cualquier duda sobre privacidad o proteccion de datos, puedes escribir a dcarrion.desarrollo@gmail.com.

2. Que datos puede tratar esta web

Con caracter general, este sitio web no requiere crear una cuenta ni enviar datos personales para navegar por sus paginas.

  • Datos que introduces voluntariamente en el formulario de contacto: nombre, correo electronico, app relacionada y mensaje.
  • Preferencia de idioma almacenada localmente en tu navegador para recordar si prefieres espanol o ingles.

En este momento la web de Simpli no activa Google Analytics ni herramientas de medicion equivalentes.

3. Finalidades y base juridica

  • Atender consultas, incidencias, sugerencias o mensajes enviados desde el formulario de contacto.
  • Gestionar la comunicacion contigo si solicitas una respuesta.
  • Recordar localmente tu idioma preferido para mejorar la experiencia de navegacion.

La base juridica del tratamiento en el formulario es tu consentimiento al enviar voluntariamente tus datos. En el caso de la preferencia de idioma, la base es el interes legitimo en ofrecer una experiencia basica de navegacion coherente.

4. Proveedores y destinatarios

El formulario de contacto utiliza Web3Forms como proveedor tecnico para recibir y procesar el envio del mensaje. Eso implica que los datos del formulario se transmiten a dicho servicio para que el mensaje llegue al correo de contacto.

Fuera de ese supuesto, Simpli no vende tus datos ni los cede con fines publicitarios.

5. Conservacion

Los mensajes recibidos a traves del formulario y del correo electronico se conservaran durante el tiempo necesario para atender tu solicitud y, en su caso, para mantener un historial razonable de soporte o seguimiento.

6. Derechos

Puedes solicitar acceso, rectificacion, supresion, limitacion u oposicion respecto de tus datos personales escribiendo a dcarrion.desarrollo@gmail.com. Si consideras que el tratamiento no es correcto, tambien puedes presentar una reclamacion ante la autoridad de control competente, como la Agencia Española de Proteccion de Datos.

7. Cambios en esta politica

Esta pagina puede actualizarse si cambia el funcionamiento real del sitio web, del formulario de contacto o de los servicios utilizados para gestionarlo.
