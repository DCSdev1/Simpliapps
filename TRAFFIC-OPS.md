# Simpli Traffic Ops

## Weekly panel
- Search Console impressions
- Search Console clicks
- Search Console CTR
- Top non-brand queries
- Top landing pages
- mySupli App Store clicks from web
- myDose landing visits

## Weekly workflow
1. Publish one new piece or update one existing page.
2. Add at least one internal link from:
   - home, apps, novedades, or an app landing
3. Request indexing in Search Console for the updated URL.
4. Publish one post on X or LinkedIn.
5. If the topic fits, publish one useful comment/post in a relevant community.

## If metrics move slowly
- High impressions, low CTR:
  rewrite title and meta description
- Indexed, no impressions:
  improve intent targeting and internal links
- Clicks but low conversion:
  strengthen CTA path toward app landing or App Store
